# 小米盒子 3 增强版救砖并获取 Root 权限
### 没有目录
最简单的一次刷机。但也是踩坑比较多的一次。
使用 Kingroot 便可直接获取到盒子的 Root 权限。若要更换 Root 管理器，可以用脚本一键完成替换。
要注意的点: 1. 若刷入 ATV / GAPPS ，需在执行脚本后进入 Recovery 模式清除盒子数据。若没有合适的办法进入 Rec ，请不要进行这部分操作，有可能导致设备卡二屏无法进入系统。
2. 若设备卡一屏或二屏，且因为遥控器或其它问题无法进入 Rec ，可以使用 SP_Flash_Tool 刷入线刷包进行修复。若仍然出现问题，则为 Emmc 损坏，需要进行更换。
