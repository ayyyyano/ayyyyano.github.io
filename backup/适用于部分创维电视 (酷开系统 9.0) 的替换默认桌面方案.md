# 酷开系统 9.0 替换默认桌面
### 前言
此方案仅在创维 **65G22** (系统版本 9.0.30208-302080 / Android 10 ) 上测试通过，仅供参考，不保证其它电视的可用性。
本文仅为个人经验分享，且时效性较强，请谨慎尝试。因操作不当而出现的问题这里一概不负责。

### 准备工作
需要可正常使用的电视与遥控器，以及一部安卓版本高于 5.0 的手机。
所需软件及详细操作图: [下载](https://www.123865.com/s/kCTTjv-T4V8H)

### 操作步骤
![Image](https://github.com/user-attachments/assets/eec6f716-fc9f-46a7-8044-5c7cf3cd438d)

1. 进入电视设置，依次进入通用 -> 关于本机 -> 本机信息 -> 更多信息。进入后用遥控器依次按下 **上上下下左右左右** (或 **上上下下左左右右** )，进入工厂菜单。

2. 进入工厂菜单，进入通用设置。在通用设置中即可找到 ADB 开关，将其打开。若要关闭开机广告，打开通用设置中的 **单键开关** 以及 **快速演示模式开关** 即可。

3. 在手机上安装 **甲壳虫 ADB 助手** (可见链接或自行下载) 。将手机与电视连接到同一网络下。在软件中点击搜索图标查找电视并连接。连接后，进入甲壳虫 ADB 助手的 **应用管理** 中给电视安装 **Projectivy Launcher** (其它桌面请自行测试) 以及 **应用管家** 。

> 也可在此根据自身需要冻结部分系统应用，可在网盘链接中查看我冻结的应用，但请不要冻结 **主页** ，可能会导致系统启动卡在 **第二屏** ，需要按机身待机键 **恢复出厂设置** 才能修复。如果实在看不惯，冻结主页后请将设置 -> 信号源中的开机启动改为信号源，但此操作风险请自行承担。


4. 在应用管理中右上角刷新应用列表，启动应用管家。在应用管家中的 **无障碍服务** 中授予 Projectivy Launcher 与应用管家**无障碍权限**。在 **自动化设置** 中添加 **按键映射** 和 **事件映射** 。事件映射当启动主页时改为为返回上一应用 + 启动 Projectivy Launcher ，按键映射将主页键改为启动 Projectivy Launcher 。

5. 启动 Projectivy Launcher 。在桌面设置 -> 常规中启用 **覆盖当前启动器** 设置。在桌面设置 -> 电源中设置 **开机时自动启动** 为 Projectivy Launcher 。

- [x] 经过上述操作后，Projectivy Launcher 便实现了开机自启动，GITV 主页启动会被拦截从而跳转到安装的桌面，遥控器主页键功能也已经被取代。若需切换/查看信号源，请启动桌面上的 **CoocaaTV** 。
